CSS的display & 块级元素和行内元素

display =

  1. display = none
    元素被隐藏.此时元素脱离文档普通流,不占据空间.
    • 与visibility = hidden的区别
      1. visibility =hidden只是让元素不可见,仍占据位置,仍在文档普通流中.
      2. 改变display会导致reflows从而也导致repaints.改变visibility只会导致repaints
      3. display为非继承属性;visibility为可继承属性.
  2. display = block
    元素以块级显示.块级元素的前后都有换行符
    • 块级元素会自动继承父元素的不影响DOM树只影响渲染树的属性.
    • 子元素如果没有设置width属性,那么与父元素同宽,但这不是继承,是块级元素本身要占据最大空间的性质.
  3. display = inline
    将元素设置为行内元素,可以在同一行显示.
    • 行内元素无法设置height/width属性,根据内容选择.
      • 解决方法:
        使用line-height,计算line-height与font-size的差值,分为两半加到一个文本行内容的顶部和底部.
    • 行内元素不可设置外上下边距,可以设置外左右边距;可以设置所有内边距.
  4. display = inline-block
    将元素显示在行内,但可以设置其上下外边距,可以设置height/width.
  5. display = inherit
    继承父元素display属性.

box-sizing =

  1. content-box : 宽高应用到元素的内容框,在宽高之外绘制内边距/边框.
  2. border-box : 宽高包括了内边距和边框,在其内部绘制内容.
  3. inherit : 继承.

position =

  1. absolute : 生成据对定位的元素,相对于第一个static以外的父元素进行定位,通过left/right/top/bottom属性进行规定.
  2. fixed : 生成绝对定位的元素,相对于浏览器窗口进行定位,通过left/right/top/bottom属性进行规定.
  3. relative : 生成相对定位的元素,相对于其他正常位置进行定位.
  4. static : 默认值.没有定位,元素出现在正常的流中.(忽略top/bottm/left/right).
  5. inherit : 继承.

块级元素/行内元素 — block element / inline element

  • 块级元素的特点 :
    • 每个块级元素独占一行(float除外).
    • 可以设置height/width/margin/padding.
    • 宽度默认占满容器.
    • 可以容纳其它块级元素和内联元素
  • 内联元素的特点 :
    • 和其它元素在同一行.
    • 不能设置height/width,margin和padding只能设置左右.
    • 只能容纳其它内联元素.
  • 常见块级元素 :
    div / dl / p / ul / h1-6 / form/ ol / menu / table
  • 常见内联元素
    a / br / input / img / lable / span / select
