Javascript中的NEW & CALL()

NEW

  1. 创建一个新的对象
  2. this变量指向该对象
  3. 空对象的原型执行函数的原型
  4. 改变构造函数内部this的指向

CALL()

  • 可以让call()中的对象调用当前对象拥有的function.
  • 返回值是你调用的方法的返回值,若该方法没有返回值则返回undefined.
    示例 :
  • 使用call()方法调用父函数的构造函数.
  • 使用call()方法调用匿名函数.
  • 使用call()方法调用函数并且指定上下文的this.
